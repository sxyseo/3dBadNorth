# Bad North 3D

一个受 Bad North 启发的 3D 实时战略岛屿防御游戏。

## 游戏特色

- **3D 实时战略**: 在 3D 环境中指挥你的小队进行战斗
- **岛屿防御**: 保护岛屿免受敌人入侵
- **RTS 风格控制**: 使用经典 RTS 操作方式选择和指挥单位
- **波次系统**: 抵御逐渐增强的敌人波次
- **经济系统**: 管理金币，招募新单位
- **程序化岛屿**: 动态生成的岛屿地形

## 游戏玩法

### 基本控制

- **左键点击**: 选择单位
- **左键拖拽**: 框选多个单位
- **Shift + 点击**: 多选单位
- **右键点击**: 移动/攻击命令
- **ESC**: 取消选择
- **WASD**: 移动摄像机
- **鼠标滚轮**: 缩放
- **鼠标中键拖拽**: 旋转视角
- **空格键**: 开始下一波

### 游戏机制

1. **招募单位**: 花费金币招募新的战斗单位
2. **防御波次**: 敌人会分批进攻岛屿
3. **保护基地**: 不要让所有单位阵亡
4. **赚取金币**: 击杀敌人和完成波次获得金币奖励
5. **升级天数**: 完成所有波次后进入下一天

## 技术栈

- **引擎**: Unity 2022.3 LTS
- **语言**: C#
- **架构**: 组件化设计
- **AI**: NavMesh 寻路
- **UI**: Unity UI (TextMeshPro)

## 项目结构

```
Assets/
├── Scripts/
│   ├── Managers/          # 游戏管理器
│   ├── Player/            # 玩家单位
│   ├── Enemies/           # 敌人 AI
│   ├── Combat/            # 战斗系统
│   ├── Islands/           # 岛屿生成
│   ├── UI/                # 用户界面
│   └── Utilities/         # 工具类
├── Prefabs/               # 预制体
├── Scenes/                # 场景文件
├── Materials/             # 材质
└── Models/                # 3D 模型
```

## 核心系统

### 1. GameManager
游戏主控制器，管理游戏流程、波次、资源等。

### 2. UnitSelectionManager
处理单位选择和 RTS 风格控制。

### 3. SquadUnit
玩家控制的战斗单位，支持移动、攻击、自动战斗。

### 4. Enemy
敌人 AI，自动寻找并攻击玩家单位。

### 5. IslandGenerator
程序化生成岛屿地形。

### 6. UIManager
管理所有 UI 元素和用户交互。

## 安装和运行

1. 安装 Unity 2022.3 LTS 或更高版本
2. 打开项目文件夹
3. 打开 `Assets/Scenes/MainMenu.unity` 或 `GameScene.unity`
4. 点击 Play 开始游戏
