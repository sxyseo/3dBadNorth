# Unity测试检查清单

## 🎯 快速开始

### 1. 使用编辑器工具
在Unity菜单栏执行：
```
Bad North 3D > Quick Setup > Create All
```

### 2. 手动设置（如果自动工具失败）
- 创建地面和光照
- 生成预制体
- 设置GameManager引用
- 烘焙NavMesh

### 3. 测试核心功能
- 单位选择和移动
- 敌人生成和AI
- 波次系统
- 音频系统
